# Marketing Campaign Analysis

## Project Overview
This project analyzes customer behavior and marketing campaign performance to identify high-value customer segments and improve marketing ROI.

## Tools Used
Python
Pandas
NumPy
Seaborn
Matplotlib

## Dataset
Marketing Campaign Dataset (Kaggle)

## Key Insights
- High income customers generate the majority of revenue
- Customers with recent purchases are more likely to respond to campaigns
- Store purchases dominate compared to web purchases

## Business Recommendations
- Focus campaigns on high-spending customers
- Target customers with recent purchase history
- Increase digital marketing strategies
